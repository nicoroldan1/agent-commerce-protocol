# TAN — Glosario y definiciones

- **TAN (Tienda Agent‑Native)**: plataforma e‑commerce operable por agentes.
- **ACE (Agent Commerce Endpoint)**: API estándar para que agentes consulten y compren.
- **Policy Engine**: componente que decide si una acción está permitida, denegada o requiere aprobación.
- **Approval**: control humano o automatizado para acciones sensibles.
- **Audit Log**: registro inmutable de acciones y cambios.
- **Connector/Adapter**: integración con sistemas externos (pagos, envíos, ads).
- **Budget**: límites de gasto/uso por store/agente.
